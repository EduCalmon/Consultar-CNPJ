# Consultar-CNPJ
Faz consulta de CNPJ por meio da Brasil API. Tem o programa completo e também apenas o código simples que devolve as informações em json da API.
O código é bem simples. Também usei um dicionário para organizar o que a API devolvia ao ser chamada. Tem algumas coisas que não coloquei para mostrar, que não são
tão importantes na hora de consultar um CNPJ (isso para mim). Mas se quiser ver tudo, o arquivo "teste-api.py" Mostra tudo que a API devolve. Obrigado!
